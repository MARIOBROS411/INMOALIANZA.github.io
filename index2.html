Instrucciones de Uso

  1. Agrega este código a tu sitio web - Inserta este verificador en tu página donde tienes el Facebook Pixel instalado.
  2. Haz clic en los botones - Simula las acciones que quieres rastrear (pagos, clics en video, etc.).
  3. Verifica los registros - Observa si los eventos aparecen en el panel de registro.
  4. Inspecciona la consola - Abre las herramientas de desarrollador (F12) y ve a la pestaña "Console" para más detalles.
// Código para verificar eventos manualmente (ya incluido en esta página) function trackFacebookEvent(eventName, parameters) { console.log(`[Facebook Pixel Simulator] Evento: ${eventName}`, parameters); // Registrar en la interfaz logEvent(eventName, parameters); // Si Facebook Pixel está disponible, enviar el evento real if (typeof fbq !== 'undefined') { fbq('track', eventName, parameters); console.log(`[Facebook Pixel Real] Evento enviado: ${eventName}`); } }
